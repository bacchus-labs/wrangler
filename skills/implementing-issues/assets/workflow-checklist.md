# Feature Implementation Checklist

Copy this checklist and track your progress:

## Setup
- [ ] Parse scope (spec/plan/issues)
- [ ] Create todo tracking
- [ ] Capture working directory
- [ ] Build dependency graph

## For Each Task
- [ ] Mark task in progress
- [ ] Dispatch implementation subagent
- [ ] Verify location verification passed
- [ ] Verify tests passing
- [ ] Verify TDD certification provided
- [ ] Dispatch code reviewer
- [ ] Fix Critical issues (2 attempts max)
- [ ] Fix Important issues (2 attempts max)
- [ ] Document Minor issues
- [ ] Mark task complete
- [ ] Update dependencies

## Final Verification
- [ ] Run full test suite
- [ ] Verify all requirements met
- [ ] Aggregate TDD certifications
- [ ] Aggregate code reviews
- [ ] Check git status clean

## Completion
- [ ] Present summary to user
- [ ] Invoke finishing-a-development-branch
